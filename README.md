
# Công cụ sử dụng

## ReactJS + Vite
ReactJS được sử dụng để xây dựng giao diện người dùng. Vite đóng vai trò là công cụ build và môi trường phát triển cho React.

## Django
Django là framework back-end chính của hệ thống, chịu trách nhiệm xử lý các logic, xây dựng REST API và quản lý quy trình trao đổi dữ liệu giữa front-end và database.

## MySQL
MySQL được sử dụng làm hệ quản trị cơ sở dữ liệu., tích hợp với Django thông qua ORM.
